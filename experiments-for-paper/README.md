# Simulations for 2024 paper

This folder contains code for the simulations in our 2024 paper on inverse-variance weighted treatment effects. 
The file `wpor_sim_rslurm.R` is meant to be run on a slurm cluster, followed by `plot_results.R`.

Within `wpor_sim_rslurm.R`, there is a deactivated workspace chunk that can be used 
to manually run, test and/or debug a handful of simulations on a local machine, if using R interactively.
